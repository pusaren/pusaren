Ongoing Project: Screw Collecting Robot

Currently, I'm working on an exciting project: a **Screw Collecting Robot**. This robot is designed to automate the collection of screws and small metal parts in industrial environments. The project involves integrating various sensors, actuators, and a smart control system to make the robot efficient in navigating and identifying scattered screws.

The main challenge of this project is developing a precise identification system using computer vision and machine learning, along with ensuring the robot can move autonomously in a dynamic workspace. Once completed, this project aims to improve efficiency and reduce manual labor in industrial applications.
